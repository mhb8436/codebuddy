# CodeBuddy 통합 테스트 시나리오

## 테스트 환경

- 클라이언트: http://localhost:3004
- 서버 API: http://localhost:4000
- 테스트 계정: admin@codebuddy.local / admin1234

---

## 시나리오 1: 학습 흐름 전체 테스트

### 1.1 로그인 및 메인 페이지

1. http://localhost:3004 접속
2. 로그인 화면 표시 확인
3. 테스트 계정으로 로그인
4. **메인 페이지 확인**:
   - "어떤 언어를 배우고 싶으신가요?" 타이틀 표시
   - JavaScript, Python 언어 카드 표시
   - (이전 학습이 있는 경우) "이어서 학습하기" 카드 표시

### 1.2 언어 → 트랙 → 토픽 선택

1. **JavaScript 선택**
   - URL: `/?lang=javascript`로 변경
   - "학습 트랙을 선택하세요" 화면 표시
   - JavaScript 입문, JavaScript 중급 트랙 표시
   - "< 언어 선택으로 돌아가기" 버튼 표시

2. **JavaScript 입문 트랙 선택**
   - URL: `/?lang=javascript&track=js-beginner`로 변경
   - "JavaScript 입문" 타이틀과 토픽 목록 표시
   - 스테이지별 토픽 그룹 표시 (기초 문법, 함수, 배열, 객체)
   - "< 트랙 선택으로 돌아가기" 버튼 표시

3. **변수와 상수 토픽 선택**
   - URL: `/learn`으로 이동
   - 학습 페이지 표시

### 1.3 학습 페이지 (LearningPage)

1. **Breadcrumb 네비게이션 확인**:
   - "JavaScript > JavaScript 입문 > 변수와 상수" 형태로 표시
   - "JavaScript"는 파란색 링크 (클릭 가능)
   - "JavaScript 입문"은 파란색 링크 (클릭 가능)
   - "변수와 상수"는 현재 위치 (검은색, 클릭 불가)

2. **4개 탭 확인**:
   - 개념 학습 탭 (기본 선택)
   - AI 튜터 탭
   - 연습문제 탭
   - 시험 탭

3. **개념 학습 탭 내용**:
   - 개념 카드들 표시 (let과 const, 기본 자료형 등)
   - 각 카드에 키워드 태그와 예시 코드 표시

## 시나리오 2: Breadcrumb 네비게이션 테스트

### 2.1 학습 페이지 → 토픽 선택 화면

1. `/learn` 페이지에서 시작
2. Breadcrumb에서 **"JavaScript 입문"** 클릭
3. **예상 결과**:
   - URL: `/?lang=javascript&track=js-beginner`
   - 토픽 선택 화면 표시
   - JavaScript 입문 트랙의 모든 토픽 목록 표시

### 2.2 학습 페이지 → 트랙 선택 화면

1. `/learn` 페이지에서 시작
2. Breadcrumb에서 **"JavaScript"** 클릭
3. **예상 결과**:
   - URL: `/?lang=javascript`
   - 트랙 선택 화면 표시
   - JavaScript 입문, JavaScript 중급 트랙 표시

## 시나리오 3: 뒤로가기 버튼 테스트

### 3.1 토픽 선택 → 트랙 선택

1. `/?lang=javascript&track=js-beginner` (토픽 선택 화면)
2. **"< 트랙 선택으로 돌아가기"** 버튼 클릭
3. **예상 결과**:
   - URL: `/?lang=javascript`
   - 트랙 선택 화면 표시

### 3.2 트랙 선택 → 언어 선택

1. `/?lang=javascript` (트랙 선택 화면)
2. **"< 언어 선택으로 돌아가기"** 버튼 클릭
3. **예상 결과**:
   - URL: `/`
   - 언어 선택 화면 표시 ("어떤 언어를 배우고 싶으신가요?")

## 시나리오 4: URL 직접 접근 테스트

### 4.1 트랙 선택 화면 직접 접근

1. 브라우저에서 `http://localhost:3004/?lang=javascript` 직접 입력
2. **예상 결과**:
   - 트랙 선택 화면 바로 표시
   - JavaScript 입문, JavaScript 중급 트랙 표시
   - 뒤로가기 버튼 정상 동작

### 4.2 토픽 선택 화면 직접 접근

1. 브라우저에서 `http://localhost:3004/?lang=javascript&track=js-beginner` 직접 입력
2. **예상 결과**:
   - 토픽 선택 화면 바로 표시
   - JavaScript 입문 트랙의 토픽 목록 표시
   - 뒤로가기 버튼 정상 동작

---

## 시나리오 5: 이어서 학습하기 테스트

### 5.1 학습 선택 후 메인으로 이동

1. JavaScript → JavaScript 입문 → 변수와 상수 선택하여 `/learn` 진입
2. 로고(CodeBuddy) 클릭하여 메인 페이지(`/`)로 이동
3. **예상 결과**:
   - "이어서 학습하기" 카드 표시
   - "JavaScript · JavaScript 입문 · 변수와 상수" 텍스트 표시
   - "계속하기" 버튼 표시

### 5.2 이어서 학습하기 클릭

1. 메인 페이지에서 "이어서 학습하기" 카드의 **"계속하기"** 버튼 클릭
2. **예상 결과**:
   - URL: `/learn`
   - 이전에 선택했던 토픽(변수와 상수)의 학습 페이지 표시

---

## 시나리오 6: 헤더 네비게이션 테스트

### 6.1 수준 변경

1. 헤더의 "수준" 드롭다운에서 "초초보" → "조금아는초보" 변경
2. **예상 결과**:
   - 수준 변경 API 호출
   - AI 튜터 탭에서 해당 수준에 맞는 응답 제공

### 6.2 관리자 메뉴 (admin/instructor 전용)

1. admin 계정으로 로그인
2. 헤더에 "관리" 링크 표시 확인
3. "관리" 클릭 시 `/admin` 페이지로 이동 확인
